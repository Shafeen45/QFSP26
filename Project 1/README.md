## Overview
This project applies the Markowitz Mean-Variance framework to construct an optimal portfolio from 7 NYSE-listed equities. The efficient frontier is calculated using both a Monte Carlo simulation and a Machine Learning polynomial regression method.

## Assets Selected
| Ticker | Company | Sector |
|--------|---------|--------|
| AAPL | Apple Inc. | Technology |
| TSM | Taiwan Semiconductor | Semiconductors |
| XOM | ExxonMobil | Energy |
| PG | Procter & Gamble | Consumer Staples |
| JPM | JPMorgan Chase | Financials |
| LLY | Eli Lilly | Healthcare |
| CAT | Caterpillar Inc. | Industrials |

## Results
- **Expected Annual Return:** 36.22%
- **Annual Risk (Std Dev):** 13.86%
- **Sharpe Ratio:** 2.25
